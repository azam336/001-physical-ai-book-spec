# Community

Welcome to the Physical AI learning community! Connect with other learners, share your projects, and contribute to the book.

## Discussion & Questions

### GitHub Discussions

Our primary community hub is [GitHub Discussions](https://github.com/physical-ai-book/physical-ai-book/discussions).

**Categories**:

- **Q&A**: Ask questions about book content or exercises
- **Show and Tell**: Share your projects and modifications
- **Ideas**: Suggest improvements or new content
- **General**: Everything else

### Getting Help

Before asking a question:

1. Check the [Troubleshooting](./troubleshooting) guide
2. Search existing discussions
3. Review the relevant chapter section

When asking questions:

- Be specific about what you're trying to do
- Include your code and error messages
- Mention your operating system and Python version

## Contributing

We welcome contributions from the community!

### Ways to Contribute

| Contribution | Difficulty | Impact |
|--------------|------------|--------|
| Fix typos | Easy | Small |
| Improve explanations | Medium | Medium |
| Add code examples | Medium | High |
| Create diagrams | Medium | High |
| Write new exercises | Hard | High |
| Translate content | Hard | High |

### Getting Started

1. Read the [Contributing Guidelines](../contributing)
2. Fork the repository
3. Pick an issue labeled `good first issue`
4. Submit a pull request

### Recognition

Contributors are recognized in several ways:

- Listed in the repository's `CONTRIBUTORS.md`
- Mentioned in release notes
- Credited in the content they improved

### Project Ideas

Looking for inspiration? Try extending the book's exercises:

**Chapter 1 Extensions**:
- Compare PyBullet to MuJoCo or Isaac Sim
- Create a custom robot model in URDF
- Build a simple teleoperation interface

**Chapter 2 Extensions**:
- Implement SLAM using simulated LIDAR
- Add camera-based object recognition
- Fuse multiple sensor modalities

**Chapter 3 Extensions**:
- Control a 7-DOF robot arm
- Implement trajectory optimization
- Build a grasping pipeline

**Chapter 4 Extensions**:
- Train a walking policy with reinforcement learning
- Implement behavior trees
- Create a multi-robot coordination system

**Chapter 5 Extensions**:
- Build a safety monitoring dashboard
- Implement a risk assessment tool
- Create an ethics case study collection

### Learning Resources

**Books**:
- *Probabilistic Robotics* by Thrun, Burgard, Fox
- *Modern Robotics* by Lynch and Park
- *Robot Modeling and Control* by Spong, Hutchinson, Vidyasagar

**Online Courses**:
- Coursera: Robotics Specialization (UPenn)
- MIT OpenCourseWare: Introduction to Robotics
- Stanford CS231A: Computer Vision

**Simulators**:
- [PyBullet](https://pybullet.org/) - Used in this book
- [MuJoCo](https://mujoco.org/) - High-fidelity physics
- [Isaac Sim](https://developer.nvidia.com/isaac-sim) - Photo-realistic
